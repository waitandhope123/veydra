# üß± **2. Mounting Interfaces**

### **2.1 Mechanical Mounting Interface (MMI)**  
A standardized pattern of:
- Hardpoints  
- Load-bearing rails  
- Alignment pins  
- Quick-lock anchors  

MMI ensures that every module attaches:
- Securely  
- Repeatably  
- Without complex tools  
- Without compromising crash performance  

### **2.2 Electrical & Data Interface (EDI)**
A single multi-pin interface providing:
- 12 V accessory power  
- High-speed data link  
- Optional HV connection (for powered modules)  
- Safety interlocks  

Modules can report:
- Status  
- Diagnostics  
- Configuration data  

Veydra automatically recognizes installed modules.
